1. Directory Structure Overview
==========================================================================
\bin	application exe, dll
\Data	resources needed to run the exe
	notes: Currently only zhCN version mpq files are tested. In order to run the application,
	you need to copy 
	world.mpq, 
	expansion1.mpq, 
	expansion2.mpq, 
	expansion3.mpq, 
	expansion4.mpq, 
	interface.mpq, 
	model.mpq, 
	texture.mpq, 
	itemtexure.mpq, 
	misc.mpq, 
	sound.mpq, 
	wow-update-base-16016.mpq, 
	wow-update-base-16048.mpq, 
	wow-update-base-16057.mpq to \Data\ folder and 
	localed-zhCN.mpq to \Data\zhCN\ folder.
